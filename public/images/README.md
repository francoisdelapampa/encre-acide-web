## Conseils

- Format recommandé : **JPG** ou **WebP** (meilleure compression)
- Taille optimale : **1200×1680px** (ratio 5:7 des posters)
- Poids cible : **< 300KB par image** pour un chargement rapide
- Nom de fichier : correspond au `slug` du produit dans `src/lib/data.ts`

Une fois les images déposées, dis-moi et je mettrai à jour les chemins dans le code.
